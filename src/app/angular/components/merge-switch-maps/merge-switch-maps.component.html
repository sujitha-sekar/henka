MergeMap and SwitchMap are both RxJS operators used for transforming and flattening observables, but they behave differently in how they handle multiple subscriptions

Behavior: Subscribes to each inner observable without canceling the previous one, allowing all emitted observables to run concurrently.

Use Case: Perfect for handling multiple API calls where each request depends on the result of a previous one, allowing you to process all emissions concurrently.
