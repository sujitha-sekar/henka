• AngularJS (2010):

    Release Date: AngularJS, also known as Angular 1.x was released in 2010 by Google.

    It was created by Misko Hevery and a team at Google to simplify building dynamic web applications. AngularJS became one of the most popular frameworks for building SPAs(Single Page Applications). A flexible architectural pattern, often referred to as MVC (Model-View-Controller) but allowing variations.

  • Angular 2 (2016):

    Release Date: Angular 2 was released in September 2016.

    Angular 2 was a complete rewrite of AngularJS, and it was a major shift in how developers built Angular applications. This new version was referred to as simply "Angular", while the original version became known as AngularJS.

  • Angular 4 (2017):

    Release Date: March 2017.

    Google skipped Angular 3 to avoid confusion due to misaligned version numbers of core libraries. Instead, they released Angular 4.

  • Angular 5 (2017):

    Release Date: November 2017.

    Angular 5 was a smaller update focused on improving the performance and developer experience.

  • Angular 6 (2018):

    Release Date: May 2018.

    Angular 6 introduced updates that focused on making it easier to develop and maintain Angular applications with tools like the Angular CLI.

  • Angular 7 (2018):

    Release Date: October 2018.

    Angular 7 focused on improving performance and providing better development tools, especially in terms of UI components.

  • Angular 8 (2019):

    Release Date: May 2019.

    Angular 8 marked the beginning of some exciting changes for the framework, with a major focus on Ivy, the new Angular rendering engine.

  • Angular 9 (2020):

    Release Date: February 2020.

    Angular 9 was the milestone where Ivy became the default rendering engine for all Angular applications.

  • Angular 10 (2020):

    Release Date: June 2020.

    Angular 10 introduced relatively fewer features, focusing more on maintaining stability, cleaning up code, and improving existing functionality.

  • Angular 11 (2020):

    Release Date: November 2020.

    Angular 11 improved build speed and developer productivity.
